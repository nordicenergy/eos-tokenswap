## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ NETWORK=<eos/jungle/local> yarn run dev

# build for production and launch server
$ NETWORK=<eos/jungle/local> yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
